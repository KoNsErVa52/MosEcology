Данный проект создан в рамках проектной деятельности на основе МГТУ им. Баумана в 2026 году. Он представляет собой информационный ресурс, посвященный ключевым экологическим проблемам Москвы и их актуальным решениям. Сайт предоставялет возможность ознакомиться с экологическими терминами, существующими в городе эко-инициативами и сообществами, а также включает в себя развлекательно-образовательный раздел. Проект продолжает развиваться.
